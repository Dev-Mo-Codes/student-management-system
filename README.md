# 🎓 Student Management System

A command-line Student Management System built with Python using Object-Oriented Programming (OOP).

This project is part of my journey to becoming a Backend Software Engineer. It focuses on writing clean, maintainable, and scalable code while applying software development best practices.

---

## ✨ Features

- ✅ Add new students
- ✅ Remove students
- ✅ Update student information
- ✅ Search students by ID
- ✅ Search students by name
- ✅ Display all students
- ✅ Sort students by name
- ✅ Sort students by age
- ✅ Sort students by student ID
- ✅ Show total number of students
- ✅ Display average student age
- ✅ Display youngest and oldest student
- ✅ Delete all students
- ✅ Input validation
- ✅ Duplicate student ID prevention
- ✅ JSON data persistence
- ✅ Object-Oriented architecture

---

## 🏗️ Architecture

The project follows an Object-Oriented Programming design.

### Student
Represents a single student and is responsible for:
- Storing student information
- Displaying student details
- JSON serialization/deserialization

### StudentManager
Manages the collection of students and handles:
- CRUD operations
- Searching
- Sorting
- Validation
- Statistics
- Data persistence

### Main
Provides the command-line interface (CLI) and interacts with the user.
