# 🎓 Student Management System

A simple command-line Student Management System built with Python.

This project is part of my journey to becoming a Backend Software Engineer. It focuses on writing clean, maintainable code while learning software development best practices.

---

## ✨ Features

- ✅ Add new students
- ✅ Remove students
- ✅ Update student information
- ✅ Search students by ID
- ✅ Search students by Name
- ✅ Display all students
- ✅ JSON data persistence
- ✅ Input validation
- ✅ Duplicate Student ID prevention
- ✅ Sort students by Name
- ✅ Sort students by Age
- ✅ Sort students by Student ID
- ✅ Show total number of students
- ✅ Display average student age
- ✅ Display youngest and oldest student
- ✅ Delete all students

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Dev-Mo-Codes/student-management-system.git
```

Go to the project directory

```bash
cd student-management-system
```

Run the project

```bash
python main.py
```
